#### Installation
```shell
npm install -g passlawl
```

#### Usage
Generate random 4 word password in english (default)
```shell
passlawl
```

Genereate random 4 word password in norwegian (using locale)
```shell
passlawl --locale no
# Or
passlawl -l no
```
